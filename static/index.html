GlassJaw Publishing

GlassJaw Publishing is a holding company that owns multiple publishing and media production companies. We work with thousands of authors to help them produce, market and sell their work.

We are implementing a central platform that will enable our authors to monitor the interest of the public in their work and to track the royalties associated with their work. This platform is our GlassJaw Author Management.

GlassJaw Author Management Technical details

The GlassJaw Author Management is designed to be API first in order to enable our subsidiaries to build their own frontend according to their branding and needs.

At this moment the microservice API provides centralized management of royalties paid to book authors.

The platform provides the ability for the authors to register to the platform, to add their books as well as their initial royalties and to monitor the details of their royalties. To maintain security we have enabled all authors to reset their passwords whenever they need to.

User flow

Outside visitors

The visitors can see the number of users via the /users/v1 endpoint The visitors can see the total number of users/authors via the /books/v1 endpoint

Registered users

The user can register to the system via the register /users/v1/register endpoint. After registration, the user can log in to the system via the /users/v1/login endpoint. After logging in the user can get their information via the /users/v1/{username} endpoint or the /users/v1/id/{id} endpoint. After logging in the user can reset their password via the /users/v1/{username}/password endpoint After logging in the user can list their books and royalties via the /books/v1/user/{username} After logging in the user can register another of their books via the /books/v1 endpoint.

Session length - variable, check what is configured on the platform.
